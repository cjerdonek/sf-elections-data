SF Elections Data

SF Elections Data is a project to expand the amount of open data and open source tools to work with an Francisco elections.

Current projects include--

  • SF Base Election Data (SFBED). A JSON file that captures the information about San Francisco elections that tends to remain the same from election to election.
  • WinEDS Converter. A tool to make San Francisco precinct-level vote totals available in an easy-to-parse TSV format.
  • OpenRCV. A tool to tabulate the results of ranked choice voting elections like those conducted in San Francisco. This is also a Knight News Challenge submission.

SF Elections Data is a project of Code for San Francisco.
